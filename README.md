# 小🐷生日 H5 v6

这一版继续减少解释感和“人机味”，把文案改得更轻、更像真的送给她看的生日卡片。

## 音频文件放这里

把音频放入：

```text
assets/audio/
```

按下面文件名命名：

```text
birthday-song.mp3             生日快乐歌音频
voice-1-personal.mp3          你第一段个人语音
bgm-haidilao-birthday.mp3     海底捞生日快乐歌
voice-yueding-cover.mp3       清唱《约定》
```

如果你的音频是 `.m4a`，打开 `js/content.js`，把对应后缀改成 `.m4a`。

## 女巫毒药

默认点“小猪药水”会掉出小卡片：

```text
去 QQ 聊天框发个 1
```

想换中奖药水，在 `js/content.js` 里改：

```js
potionWin: 'pig'
```

可选值：`rose` / `star` / `pig`。

## v11 手机适配修复

这一版重点修了手机端：

- 每一页都可以在手机上纵向滚动，不会再被底部按钮或屏幕高度裁掉。
- 字号、卡片内边距、蛋糕、唱片、月亮、气球等元素在小屏幕上重新缩小。
- 回忆卡片、女巫毒药、音频进度条在手机上改成更紧凑的竖排布局。
- 左右滑动翻页现在只在明确横向滑动时触发，向上/向下滚动页面不会误翻页。

建议最终在微信内置浏览器和 Safari 各测试一次。
